## 0. 서사 설계 원칙

### 0.1 위기보다 발견

이 게임에는 악당, 오염, 화산 폭발, 고갈된 수원, 마른 샘이 등장하지 않는다.

사용자가 해결할 것은 실제 수원지의 문제가 아니라 **보이지 않는 자연의 구조를 이해하는 퍼즐**이다.

> 제주삼다수는 위험에서 구해내는 물이 아니라, 긴 시간과 지층과 관리가 만들어낸 신뢰의 물이다.

**서사 구동력 — 한 방울의 동행**: 위기를 제거하면 이야기를 미는 힘이 필요하다. 그 힘은 동행이다 — 플레이어는 한라산 고지에 내린 **빗방울 하나의 31년 여정을 안내하는 동행자**다. 매 챕터의 목표는 "구한다"가 아니라 "이 방울이 다음 층에 무사히 닿게 한다"이며, 긴장은 위협이 아니라 퍼즐의 난이도와 발견의 밀도에서 나온다. 엔딩 카피("당신이 몇 분 동안 지나온 길을, 한 방울은 평균 31년 동안 걸어옵니다")는 이 구동력의 자연스러운 종착이다.

### 0.2 물은 소모품이 아니다

- 물방울은 체력 화폐가 아니다.
- 플레이어가 보호하고 안내해야 할 핵심 오브젝트다.
- 이동 자원은 `탐사력` 또는 `행동 횟수`로 분리한다.
- **탐사력 회복 아이템 = 감귤 (확정)**: 물이 아닌 자원으로 회복해야 하므로(물 상품화 회피 원칙), 보조로 강등됐던 감귤을 회복 아이템으로 복권한다. 제주 정체성 유지 + 건강 효능 오인 없음 + 원판 IP와의 자원 연속성까지 동시에 해결하며, 레벨 경제(시작 탐사력 + 회복량 × 배치수 ≥ 최단경로 × 여유율)와 솔버 검증 체계가 그대로 보존된다.
- 물을 적게 썼다는 표현 대신 정확한 경로와 균형을 평가한다.

### 0.3 정보는 행동 뒤에 온다

```
행동
→ 결과
→ 짧은 브랜드 팩트
→ 선택형 상세 카드
```

긴 설명을 먼저 보여주지 않는다. 사용자가 방금 수행한 행동의 의미를 사후에 알려준다.

### 0.4 공식 사실과 창작 은유를 분리한다

- 공식 사실: 180만 년, 해발 1,450m, 화산암반층, 지하 420m, 평균연령 31년, 관측·관리
- 게임 은유: 화산석 밀기, 투수 통로, 저류 노드, 관측 문양, 지식 결정
- “게임 속 구조가 실제 지질과 동일하다”고 주장하지 않는다.

## 1. 전체 경험 구조

사이트는 스크롤을 내릴수록 지상에서 지하로 이동한다. 게임은 이를 조작 가능한 하강 여정으로 바꾼다.

```
제주도 형성
↓
스며듦
↓
머금음
↓
청정함
↓
소중함
```

### 랜딩페이지

- 사용자가 아래로 스크롤하면 제주 단면이 열린다.
- 각 구간에서 짧은 애니메이션과 게임 진입점이 나타난다.
- “읽기”와 “플레이”가 동일한 수직 구조를 사용한다.

### 월드맵

섬 조감도가 아니라 **제주 지층 단면형 월드맵**을 사용한다.

| 위치 | 표현 |
|---|---|
| 상단 | 한라산 고지, 구름, 비 |
| 중상단 | 숲, 초지, 다공질 화산지질 |
| 중단 | 겹겹의 용암층과 퇴적층 |
| 중하단 | 암석 틈, 저류 구간, 푸른 물길 |
| 하단 | 지하 420m 취수 지층 |
| 우측 메타 영역 | 관측, 보전, 순환, 사람의 관리 |

## 2. 프롤로그 — 제주도 형성

### 브랜드 의미

좋은 물은 물이 내린 순간부터 시작된 것이 아니다. 물을 품을 수 있는 섬과 지층이 오랜 시간 형성된 결과다.

### 플레이 동사

**쌓다 · 맞추다 · 발견하다**

### 게임 구성

- 짧은 지층 조립 퍼즐
- 라봉 계열의 힘 캐릭터가 화산석을 밀어 층을 완성
- 용암층과 퇴적층의 시각적 차이를 맞추면 다음 장 개방
- 벽면의 조개 화석 이스터에그 발견 가능

### 기존 엔진 매핑

| 기존 규칙 | 브랜드 에디션 |
|---|---|
| 돌 밀기 | 화산석·지층 블록 배치 |
| 버튼 | 지층 결합점 |
| 돌문 | 오래된 암반 경계 |
| 보물상자 | 제주 형성 기록석 |

### 완료 메시지

> **물의 이야기는 비가 내리기 훨씬 전, 제주가 태어나던 시간부터 시작됐습니다.**

### 시각 방향

- 밤의 재난이 아니라 바다 속에서 섬이 태어나는 장엄한 창세 장면
- 뜨거운 주황, 현무암 회색, 깊은 바다 청색
- 폭발은 위협이 아니라 지질 형성의 자연 현상으로 짧게 표현

## 4. Chapter 2 — 머금음

### 브랜드 의미

지하수는 거대한 지하 강처럼 흐르는 것이 아니라 암석 틈 사이에 존재하며 천천히 더 깊은 곳으로 이동한다. 그 과정에서 지층은 물을 머금고 흐름을 조절한다.

### 플레이 동사

**머물다 · 균형을 맞추다 · 천천히 이동하다**

### 게임 구성

- 강 형태의 넓은 물 타일 대신 암석 사이 가는 물길과 작은 저류 공간
- 여러 `저류 노드`를 올바른 순서로 활성화
- 특정 노드가 모두 채워져야 다음 암반문이 열린다
- 거북이 계열 캐릭터가 좁은 물길을 따라 이동
- 미네랄 결정은 선택 수집물이며 클리어 필수 아님

### 기존 엔진 매핑

| 기존 규칙 | 브랜드 에디션 |
|---|---|
| 물 타일 | 암석 틈의 물길 |
| 거북이 수영 | 지하 물길 이동 |
| 버튼 그룹 | 저류 균형 노드 |
| 돌 밀기 | 흐름 조절용 화산석 배치 |
| 구덩이 메우기 | 불안정한 지층 안정화 |
| 귤 보너스 | 선택형 지질·미네랄 기록 |

### 신규 규칙 최소화

실제 유체 시뮬레이션을 만들지 않는다. 기존 버튼 그룹과 문 개방 규칙을 `머금음`의 은유로 사용한다.

### 완료 메시지

> **물은 암석의 틈에 머물며, 오랜 시간 천천히 더 깊은 곳으로 이동합니다.**

### 감각 설계

- BGM 템포 하향
- 물방울 파동과 잔향
- 빠른 타이머 없음
- 점수는 적정 순서·경로 정확도 중심

청정함은 제주 자연의 조건과 지속적인 품질 확인이 함께 만든 신뢰다.

### 플레이 동사

**확인하다 · 연결하다 · 증명하다**

### 게임 구성

- 청정 3요소를 상징하는 세 구역을 순서대로 통과
  1. 청정 고지의 빗물
  2. 화산지질
  3. 자연 상태의 주변환경
- 수질·토양 관측 노드를 연결
- 물길을 방해하지 않으면서 모든 관측점 활성화
- 최종장은 새로운 능력을 추가하지 않고 기존 세 능력의 조합만 사용

### “규칙을 더하지 않는 최종장”

고도의 처리 기술을 과장해 표현하지 않는다. 지금까지 익힌 규칙만으로 가장 정교한 퍼즐을 풀게 하여 자연이 만든 구조와 기본에 충실한 품질 관리를 게임 디자인으로 은유한다.

### 기존 엔진 매핑

| 기존 규칙 | 브랜드 에디션 |
|---|---|
| 버튼 | 관측 노드 |
| 버튼 그룹 | 수질·토양·환경 확인 세트 |
| 돌문 | 검증 완료 후 열리는 암반문 |
| 상자 | 청정 기록 아카이브 |

### 표현 주의

- 처음부터 물은 맑게 표현

### 플레이 동사

**관찰하다 · 대비하다 · 지키다 · 함께하다**

### 게임 구성

- 한정된 관측 장비를 적절한 위치에 배치

#### 엔딩 이후 메타 게임

- 개인 미션: 지식 퀴즈, 기록 도전, 재플레이

- 공병 회수·환경 프로그램 참여는 선택 인증
- 참여량이 실제 사회공헌과 연결될 경우 사전 협약과 검증 필수

| 상자 | 보전 약속 아카이브 |

### 완료 메시지

## 7. 엔딩 — 한 방울의 시간

### 엔딩 구성

1. 상단의 비가 제주 지층으로 내려간다.
2. 사용자가 지나온 스테이지가 단면 지도에 연결된다.
3. 지하 420m의 취수 지층이 빛난다.
4. 관측과 관리의 파동이 주변을 감싼다.
5. 마지막에 제주삼다수 제품이 절제되게 등장한다.

### 엔딩 카피

> **당신이 몇 분 동안 지나온 길을, 한 방울은 평균 31년 동안 걸어옵니다.**

### 제품 노출 원칙

- 게임 도중 반복 광고 금지
- 병을 체력 아이템으로 사용하지 않음
- 여정의 완성 결과로만 등장
- 구매 CTA는 엔딩 이후 별도 카드로 분리

## 8. 캐릭터 역할

캐릭터의 공식 이름은 브랜드 협의 전 확정하지 않는다.

| 능력 캐릭터 | 게임 기능 | 물길 서사 역할 |
|---|---|---|
| 힘 캐릭터 | 화산석 밀기 | 지층 구조를 읽고 길을 만든다 |
| 탐색 캐릭터 | 얕은 벽 관통 | 스며드는 틈을 발견한다 |
| 수영 캐릭터 | 물 타일 이동 | 암석 사이 물길을 따라간다 |

### 캐릭터 운영 원칙

- 현재 라봉이·금향이·청귤이는 프로토타입 캐릭터로 표기
- 정식 브랜드 에디션은 쏭이&송이 또는 캠페인 전용 캐릭터와의 역할 조정 필요
- 캐릭터가 과학적 사실을 직접 설명하기보다 행동과 짧은 대사로 안내

## 8.5 엔진 불변 계약

본 문서의 모든 재정의(공명석, 탐사력, 저류 노드, 관측 노드)는 **표시 레이어의 데이터 교체이며 규칙 엔진(rules.js)·레벨 포맷·솔버는 단 한 줄도 변경하지 않는다.** 공명석은 접촉 기폭·십자 범위·연쇄·1회성이라는 기존 폭탄 규칙과 수학적으로 동일하고, 탐사력은 기존 체력 변수의 리네임이다. "표시명은 데이터, 내부 ID는 계약"(02 레벨 포맷 문서 원칙)이 브랜드 에디션의 기술적 전제이자, 파일럿 리드타임 단축 주장의 근거다.

## 10. 비주얼 시스템

### 주조색

- 삼다수 계열 청색
- 투명한 아쿠아
- 현무암 차콜
- 제주 숲 초록
- 지층의 송이색과 주황은 보조
- UI 서체: **제주삼다수 브랜드체** (공사가 무료 상업사용으로 배포하는 전용서체) — 비용 0으로 공식성 확보, 제안 데모부터 적용

### 챕터 색 변화

| 장 | 주조색 |
|---|---|
| 형성 | 바다 청색 + 용암 주황 + 현무암 |
| 스며듦 | 숲 초록 + 비의 하늘색 |
| 머금음 | 깊은 청록 + 현무암 회색 |
| 청정함 | 밝은 아쿠아 + 흰빛 |
| 소중함 | 청색 + 자연 초록 + 사람의 따뜻한 빛 |

### 금지 비주얼

- 탁한 물
- 오염된 취수원
- 검은 폐수
- 폭발하는 한라산
- 말라붙은 수원
- 제품 병을 무기나 폭탄으로 사용
- 쓰레기를 과도하게 전면화한 공포 연출

## 12. 31년의 활용

31년은 코어 스테이지 수로 강제하지 않는다.

### 코어 콘텐츠

- 5개 테마
- 8~10스테이지
- 부담 없이 완주 가능한 캠페인

### 라이브 콘텐츠

- `31일 물길 챌린지`
- 하루 60~120초
- 기존 레벨의 조건·경로·기록 변형
- 31일 완주 시 단면형 물길 지도 제공

31일은 실제 31년을 축약 체험한다는 의미가 아니라 **오랜 기다림을 기억하기 위한 상징적 운영 장치**로 설명한다.
