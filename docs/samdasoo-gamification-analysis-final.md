# 제주삼다수 브랜드 게이미피케이션 선제안 설계 v2

**문서 성격**: 최종안 (v1 Claude → v2 Codex → v3 Claude 검수 통합)  
**목적**: 기존 감귤섬 삼총사 퍼즐 엔진을 제주삼다수 브랜드 경험으로 전환하는 선제안 구조 확정  
**핵심 원칙**: 기존 게임에 로고를 붙이는 리스킨이 아니라, 제주삼다수의 브랜드 진실을 플레이 규칙으로 번역한다.

---

## 0. 최종 판단

제안의 본질은 “삼다수 미니게임 제작”이 아니다.

> **읽던 삼다수 스토리를, 직접 걷고 밀고 풀어보는 인터랙티브 브랜드 경험으로 전환한다.**

룰렛·출석·짝맞추기는 순간 접속과 보상 전달에는 효율적이지만 브랜드 고유성이 약하다. 본 제안은 퍼즐의 공간·규칙·자원·서사를 제주도의 형성, 스며듦, 머금음, 청정함, 소중함과 결합하여 캠페인이 종료된 뒤에도 재사용 가능한 브랜드 콘텐츠 자산을 만든다.

### 1.2 수정한 것

| 기존안 | 변경안 | 변경 이유 |
|---|---|---|

| (v2 공백 보완) 탐사력 회복 아이템 미정의 | **감귤 = 탐사력 회복 아이템 확정** | 회복 픽업은 레벨 경제·솔버 검증의 척추라 반드시 필요. 감귤은 비(非)물 자원·제주 정체성·원판 IP 연속성을 동시 충족 |

| B안 세계관 공존을 최우선 | 브랜드 스킨/엔진 제공을 기본, 공존은 옵션 | 공사가 기존 캐릭터 IP를 보유한 상황에서 제안자 캐릭터 공존은 승인 부담이 큼 |
| 임의 KPI 절대값 제시 | 파일럿 기준선 대비 상대 향상 목표 | 사전 데이터 없이 D7 25%, 구매전환 8%를 약속하면 신뢰보다 과장이 될 수 있음 |

| “무첨가 최종장” 표현 | “규칙을 더하지 않는 최종장” | 제품 효능·표시 문구처럼 오해되지 않도록 게임 디자인 언어로 제한 |

### 1.3 삭제하거나 보류한 것

- 폭탄을 “물폭탄”으로 이름만 바꾸는 방식  
  → 폭발·파괴 이미지를 유지하므로 `지층 공명석`과 파동 활성화로 시각·명칭 모두 변경한다.
- 43.5% 함양률을 핵심 연출로 사용하는 방식  
  → 과학 정보로는 유효하나 첫 캠페인에서 소비자가 손실·실패로 받아들일 수 있어 선택형 지식 카드로 보류한다.
- 미네랄을 필수 수집 목표로 만드는 방식  
  → 건강 효능 오인과 게임 복잡도 상승을 피하기 위해 선택 수집물로 한정한다.
- 구매 인증이 게임 진행을 크게 좌우하는 구조  
  → 구매자는 보너스를 얻되 비구매자도 본편을 정상 완주할 수 있어야 한다.

## 2. 제안 포지셔닝

### 2.1 대외 제안 문장

> **귀사는 스크롤로 제주삼다수의 이야기를 전하고 있습니다. 우리는 그 이야기를 걷고, 밀고, 찾아내고, 지키게 만듭니다.**

### 2.2 경쟁 구도

| 구분 | 룰렛·짝맞추기 | 제주삼다수 물길 퍼즐 |
|---|---|---|
| 참여 동기 | 즉시 보상 | 도전·발견·보상 |
| 브랜드 고유성 | 낮음 | 제주 지질·물길·보전이 규칙 자체 |
| 체류 | 수초 | 2~5분 세션 |
| 재방문 이유 | 재응모 | 신규 퍼즐·기록 개선·수집 |
| 캠페인 종료 후 | 폐기 가능성 높음 | 엔진·레벨·캐릭터·콘텐츠 재사용 |
| 데이터 | 접속·응모 | 경로·실패·재도전·완주·브랜드 카드 |
| 비용 구조 | 참여자 증가에 따라 보상비 증가 | 초기 콘텐츠 투자 후 레벨 단위 확장 |

핵심은 룰렛의 전면 부정이 아니다. 룰렛은 상단 퍼널의 즉시 참여 도구로 남겨두고, 퍼즐은 브랜드 경험과 재방문을 담당하도록 역할을 분리할 수 있다.

## 3. 브랜드 에디션 구조

### 3.1 게임명 가칭

**제주삼다수: 한 방울의 시간**

부제:

> **180만 년의 땅, 31년의 물길**

### 3.2 서사 원칙

- 실제 수원지가 위험하거나 오염됐다는 설정을 사용하지 않는다.
- 물을 “정화해야 하는 더러운 대상”으로 묘사하지 않는다.
- 화산은 폭발 위험이 아니라 물을 품는 지층을 만든 창조의 시간으로 묘사한다.
- 제품 병은 반복 수집물이 아니라 여정이 완성된 마지막 결과로 노출한다.
- 브랜드 정보는 스테이지 종료 광고가 아니라 퍼즐을 푼 이유와 연결한다.

### 3.3 핵심 플레이어 역할

플레이어는 수원지를 구하는 영웅이 아니라 **제주 물길 탐험대**다.

```
제주의 형성을 이해한다
→ 스며드는 틈을 발견한다
→ 암석 사이 물길을 연결한다
→ 청정 상태를 확인한다
→ 보전 활동에 참여한다
```

## 4. 캐릭터·IP 전략

제주삼다수는 이미 `쏭이&송이` 캐릭터를 운영하고 있으므로, 현재 삼총사를 공식 캐릭터처럼 제시하면 안 된다.

### 권장 구조

| 안 | 사용 방식 | 제안서 위치 |
|---|---|---|
| 기본안 | 게임 엔진·레벨·분석·운영 구조 제공, 캐릭터와 UI는 브랜드 에디션으로 공동 설계 | 메인 |
| 옵션 A | 기존 쏭이&송이를 안내 NPC로 사용하고, 3능력 플레이어 캐릭터는 캠페인 전용으로 공동 개발 | 1순위 옵션 |
| 옵션 B | 현재 삼총사는 기술 시연용 프로토타입으로만 사용 | 데모 설명 |
| 옵션 C | 삼총사와 삼다마을의 세계관 공존 | 브랜드가 캐릭터 확장을 원할 때만 |

IP 계약에서는 다음을 분리한다.

- **제안자 자산**: 퍼즐 엔진, 레벨 포맷, 솔버, 분석 이벤트 설계, 생성 파이프라인
- **브랜드 자산**: 제주삼다수 명칭·로고·캐릭터·공식 팩트·브랜드 에디션 아트
- **공동 산출물**: 캠페인 레벨, 지식 카드, 운영 콘텐츠

## 5. 게이미피케이션 운영 아키텍처

### 5.1 3단 콘텐츠 구조

#### 1층 — 최초 진입

- 로그인 없이 60~90초 플레이
- 한 캐릭터 능력과 한 개 브랜드 팩트
- 실패 부담이 낮은 튜토리얼
- 완료 후 앱 로그인·다음 장 진입 선택

#### 2층 — 캠페인 본편

- 5개 테마, 총 8~10스테이지
- 1회 2~4분
- 일자별 또는 주차별 순차 공개
- 지식 카드·기록·배지 수집
- 비구매자도 전체 서사 완주 가능

#### 3층 — 31일 라이브 운영

- 31년의 기다림을 기억하는 상징적 31일 챌린지
- 매일 1개의 짧은 변형 퍼즐 또는 기록 미션
- 개인 최단 경로, 주간 도전, 공동 보전 목표
- 구매 인증·공병 회수는 선택 보너스

### 5.2 채널 단계

1. **독립 모바일 웹 파일럿**  
   최소 보안·연동, QR·SNS 즉시 유입
2. **제주삼다수 가정배송 앱 웹뷰**  
   로그인·배지·쿠폰·회원 세그먼트 연동
3. **광동제약 유통 채널 온팩 QR**  
   편의점·대형마트·온라인 구매와 연결
4. **시즌 운영**  
   여름 수분, 제주 여행, 러닝, 환경교육 등 확장

## 6. 보상 설계

### 기본 원칙

게임의 재미와 성취가 1차 보상이고 경품은 2차 보상이다.

### 권장 보상

- 디지털 물길 카드
- 캐릭터 배지와 월페이퍼
- 챕터 완료 쿠폰
- 최소 이동 기록 배지
- 공동 목표 달성 시 전체 참여자 보상
- 추첨형 실물 굿즈는 제한적 사용

### 사용하지 않을 구조

- 물방울이 없으면 게임을 못 하는 강제 에너지
- 구매하지 않으면 스토리를 완주하지 못하는 구조
- 확률형 뽑기
- 과도한 일일 출석 압박
- 물을 덜 쓰는 것이 무조건 좋은 것처럼 보이는 랭킹 문구

효율 퍼즐의 점수는 `물 절약`이 아니라 `탐사 정확도`, `불필요 이동 최소화`, `물길 이해도`로 표현한다.

## 7. KPI 체계

### 7.1 퍼즐 행동

- 게임 진입률
- 첫 스테이지 완료율
- 평균 세션 시간
- 재도전율
- 챕터 완료율
- 최단 경로 도전율
- D1·D7 재방문율

### 7.2 브랜드 효과

- 스며듦·머금음·청정함·소중함 메시지 회상
- 31년 평균연령 인지
- 한라산 고지 함양과 화산암반층 연상
- 브랜드 신뢰·차별성 변화
- 지식 카드 완독률

### 7.3 앱·CRM·구매

- 웹게임 → 앱 이동률
- 로그인 전환율
- 쿠폰 다운로드·사용률
- 구매 인증 참여율
- 휴면회원 재활성률
- 게임 참여군과 비참여군의 주문 변화

### 7.4 자산성

- 신규 스테이지 제작 리드타임
- 레벨당 제작비
- 시즌 재사용률
- 운영 콘텐츠 교체 주기
- 캠페인 종료 후 재활용 가능한 에셋 비율

### 7.5 파일럿 성공 기준 설정 방법

절대 목표를 임의로 약속하지 않는다.

1. 캠페인 전 4주 앱·웹 기준선을 확보한다.
2. 룰렛 또는 일반 랜딩을 대조군으로 둔다.
3. 퍼즐 참여군의 상대 향상률을 측정한다.
4. D1·D7, 세션, 브랜드 회상, 앱 전환 중 최소 2개가 유의하게 개선될 때 확장한다.

## 11. 제안서 핵심 카피

### 브랜드 관점

> **제주삼다수의 한 모금을 광고로 설명하는 대신, 소비자가 한 방울의 긴 여정을 직접 풀어보게 합니다.**

### 디지털 관점

> **룰렛이 한 번의 클릭을 만든다면, 물길 퍼즐은 다음 접속의 이유를 만듭니다.**

### 자산 관점

> **한 번 쓰고 사라지는 미니게임이 아니라, 시즌마다 자라는 제주삼다수의 플레이어블 브랜드 세계입니다.**

### 최종 추천 한 문장

> **읽던 제주삼다수 스토리를, 풀어보는 제주삼다수 스토리로 전환합니다.**

## 12. 사실 검수 메모

- 평균연령 `31년`: **확정 사실.** 공사가 고려대 연구팀과 7년(2016~2020, 2022~2023) 추적한 안정동위원소·환경추적자 연구로 재산정, Journal of Hydrology 2025년 11월호 게재, 2025년 12월 공식 발표. "국내 생산 생수 중 최장" 표현도 공사 발표 문구.
- 과거 자료의 `18~22년`은 최신 연구·브랜드 메인 문구와 혼용하지 않는다 (구 스토리사이트·FAQ에 잔존 가능 — 제안서는 31년으로 통일).
- 신규 활용 가능 공표 수치: AI 지하수위 예측 정확도 건기 1개월 96%·3개월 72% 이상 (LSTM·GRU·ANN 앙상블, 12년 데이터) — '소중함' 지식 카드 소재.
- 모든 리네임(공명석·탐사력 등)은 표시 데이터 교체이며 rules.js·솔버 무변경 — 검증된 엔진 자산이 그대로 브랜드 에디션의 리드타임 근거가 된다.
- 공식 메인 페이지의 관측망 총수와 상세 페이지의 수위·수질·토양 관측망 수는 집계 기준이 다를 수 있으므로 JPDC 확인 전 게임에 숫자를 직접 노출하지 않는다.
- `쏭이&송이`, 유통 계약 기간, 캠페인·앱 정책은 제안 직전 최신 공식 자료로 다시 검수한다.
