# 04 — 에셋 명세 (Asset Specification): 감귤섬 삼총사

**문서 버전**: 1.0
**상위 문서**: `01-gdd.md`
**저장 위치**: `vcc-game/games/tangerine-isle/docs/design/04-asset-spec.md`
**원칙 (마스터플랜 §4 개선점 1)**: 에셋 ID가 프롬프트보다 먼저다. 05/06의 모든 프롬프트는 본 문서의 ID를 목표 파일명으로 명시하며, 생성물은 리네이밍 없이 해당 경로에 안착한다.

---

## 1. 아트 디렉션 확정값

| 항목 | 값 | 근거 |
|---|---|---|
| 스타일 | 모던 픽셀아트 (hi-bit: 픽셀 감성 + 풍부한 색·부드러운 셰이딩) | 2003 원작 계승 + 2026 감성. 헤이다와 계열 통일 |
| 타일 크기 | **128×128 px** (v3 확정 — 32px 강제 다운스케일 폐기) | ChatGPT 생성물은 고해상도 픽셀풍 일러스트. 진짜 32px로 낮추면 디테일의 99%가 파괴됨을 실측 확인. 레트로 감성은 해상도가 아니라 아트 스타일에서 나온다 |
| 룸 캔버스 | 1920×1920 논리 해상도 (15×15타일 × 128px), CSS로 반응형 축소 | 레티나·모바일에서 선명 |
| 팔레트 | 감귤 오렌지(#F28C28 계열) · 섬 초록 · 바다 청록 · 화산 재색. 마스터 팔레트는 첫 생성물에서 추출해 고정 | 색 일관성은 팔레트 고정으로만 달성된다 |
| 캐릭터 크기 | 32×32 (타일 1칸) | 그리드 정합 |

## 2. 파일 체계

```
games/tangerine-isle/assets/
├── sprites/    sheet_cat.png, sheet_rabbit.png, sheet_turtle.png
├── tiles/      sheet_terrain.png
├── objects/    sheet_objects.png
├── fx/         sheet_fx.png
├── ui/         개별 파일 (ui_*.png)
├── bg/         개별 파일 (bg_*.png)
└── audio/      bgm_*.mp3, jgl_*.mp3, sfx_*.wav
```

- 스프라이트·타일·오브젝트·이펙트는 **시트 단위 생성 → `tools/slice.js`로 절단**한다 (AI 생성은 개별 낱장보다 시트가 스타일 일관성이 압도적으로 높다).
- 시트의 셀 배치는 본 문서가 계약이다. slice.js는 아래 그리드 정의를 그대로 읽는다.

## 3. 캐릭터 시트 (3장) — 각 4열 × 3행, 셀 32×32

| 행\열 | ↓ down | ↑ up | ← left | → right |
|---|---|---|---|---|
| 행1 | walk frame A | walk A | walk A | walk A |
| 행2 | walk frame B | walk B | walk B | walk B |
| 행3 | 능력 포즈 | 능력 포즈 | 능력 포즈 | 능력 포즈 |

| 파일 | 캐릭터 | 행3 능력 포즈 | 파생 ID (slice 후) |
|---|---|---|---|
| `sheet_cat.png` | 라봉이 (흰 고양이) | 돌 밀기 (앞발 뻗음) | `spr_cat_{dir}_{a\|b\|act}` |
| `sheet_rabbit.png` | 금향이 (갈색 토끼) | 땅파기 (흙 튀김) | `spr_rabbit_{dir}_{a\|b\|act}` |
| `sheet_turtle.png` | 청귤이 (초록 거북) | 수영 (물갈퀴) | `spr_turtle_{dir}_{a\|b\|act}` |

idle = walk A 정지 프레임 재사용. 좌우 대칭이 어색하지 않은 디자인이면 left는 right 미러로 대체 가능(생성 실패 시의 폴백).

## 5. 오브젝트 시트 `sheet_objects.png` — 4열 × 2행, 셀 32×32

| 셀 | ID | 내용 |

## 10. 생성 파이프라인 요약

```
[05 프롬프트] → ChatGPT 이미지 생성 (시트/개별)
  → 다운스케일 (nearest-neighbor, 목표 픽셀 치수)
  → tools/slice.js 절단 (본 문서 그리드 계약)
  → assets/ 안착 (ID = 파일명, 리네이밍 없음)
  → 게임 코드에서 ID로 참조
```

수량 총계: 시트 6장 + UI 6파일 + 배경 6파일 = **이미지 생성 세션 약 18회**, 오디오 = Suno 8곡 + SFX 10종. "간단하게 시작" 원칙에 맞춘 최소 완결 세트다.
