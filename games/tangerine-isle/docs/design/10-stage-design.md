## 설계 원칙

스테이지마다 단 하나의 핵심 통찰이 있어야 한다. 퍼즐이 풀리는 순간은 "아, 그래서 이 돌을 여기 두면 안 됐구나"가 눈에 보이는 순간이어야 한다.

### 퍼즐 어휘 (9종)

| 자원 | 특성 |
|---|---|
| 돌 | 밀 수 있음. 버튼 유지 / 구덩이 소비 — 두 용도가 항상 경합 |
| 구덩이 | 돌을 먹으면 영구 소멸. 돌 1개가 사라진다 |
| 압력 버튼 | 돌이 올라가 있을 때만 문이 열린다. 돌을 치우면 즉시 닫힘 |
| 돌문(D) | 버튼으로만 개방, 폭탄 무효. 압력식 |
| 나무문(d) | 버튼 또는 폭탄. 폭탄으로 파괴 시 영구 개방 |
| 폭탄 | 기폭 순서 설계. 연쇄 가능. 소모 후 복구 불가 |
| 물 | 거북이 전환 잠금. 열쇠를 물 속에 두면 거북이만 획득 가능 |
| 땅굴 | 토끼가 나무 1칸 관통. 구멍이 남아 이후 동선 영향 |
| 이동 체력 | 모든 이동 = 체력 1 소모. 빠른 해법이 곧 자원 절약 |

### 좋은 퍼즐의 조건 (09 계승)

1. 위 어휘 중 **둘 이상이 동시에 경합**한다
2. 실수했을 때 **왜 막혔는지 화면에서 보인다** (눈에 보이는 소프트락)
3. 정답은 **상태 재배치**이지 단순 경로 탐색이 아니다
4. **캐릭터 순서**가 해법의 일부다

### 레이아웃 원칙 (2003 원작 계승 — 2026-07-16 추가, 11 문서 §1.4)

현재 구현된 방들은 "외곽 담장 사각형 + 빈 내부"라 밀기 동선이 단조롭다. 원작의 공간 문법을 기준으로 모든 스테이지에 아래 원칙을 적용한다.

1. **담장은 직선이 아니라 각이 진다.** 나무(T)를 계단형·L자·지그재그로 꺾어 배치한다. 담장의 모퉁이는 돌 밀기의 막다른 지점(눈에 보이는 데드락 후보)이자 우회 강제 장치다.
2. **내부 담장 클러스터.** 방 안에 2~4그루 나무 뭉치를 흩어 놓아 통행 공간이 폭 1~2칸 골목의 연속이 되게 한다. 원작의 방에는 개활지가 거의 없다.
3. **물은 대각선으로.** 물길/호수 가장자리를 계단형 대각선으로 만든다 (원작의 폭포·강). 어느 방향에서 물에 들어가고 나오는지가 선택이 된다 (1-3의 문법을 지형이 만들어 준다).
4. **돌-버튼 재사용 회귀 동선이 표준 문법.** 돌들을 버튼에 올려 한쪽 문을 열어 열쇠를 얻고, 되돌아와 같은 돌을 다른 버튼으로 재배치해 다음 문(상자)을 연다. 원작에서 확인된 원형이며 2-2, 4-1, 4-2가 이 계열이다. 챕터 2 이후 스테이지는 최소 1회의 재배치 회귀를 포함해야 한다.
5. **직사각형 검사.** 방의 통행 가능 영역이 하나의 직사각형으로 근사되면 그 방은 레이아웃 불합격. 유일한 예외는 튜토리얼 1-1이다.

### 난이도 곡선 (2026-07-16 개정)

**쉬운 구간은 챕터 1뿐이다.** 챕터 2부터는 "배우는 스테이지"가 없다.

| 챕터 | 난이도 목표 | 체력 여유율 α | 요구 조건 |
|---|---|---|---|
| 1 | 쉬움 — 어휘 습득 | ≥ 1.8 | 소프트락 없음, 힌트 허용 |
| 2 | 어려움 시작 | ≈ 1.4 | 함정 최소 1개, 재배치 회귀 1회 이상, 힌트 없음 |
| 3 | 어려움 — 영구 선택 | ≈ 1.3 | 되돌릴 수 없는 실수 경로 존재 |
| 4 | 고난이도 | ≤ 1.2 | 여유 돌 0개 (돌 수 = 필요 수), 복합 잠금 |

난이도를 올리는 표준 장치 (챕터 2 이후):
- 여유 돌 제거: 사용 가능한 돌 수를 해법 필요 수와 정확히 일치시킨다.
- 귀환 동선 조이기: 열쇠를 얻고 돌아오는 길에 닫힌 버튼문/구덩이를 배치해 회귀 타이밍을 해법의 일부로 만든다.
- 담장 모퉁이에 의한 밀기 각도 제한: 돌을 버튼까지 미는 경로가 한 방향뿐이 되도록 지형으로 강제한다.
- 힌트 오브젝트 제거.

## 챕터 구조 개요

```
챕터 1 — 감귤 과수원     룸 1×3~3×1    4스테이지   어휘 소개              쉬움
챕터 2 — 귤밭 창고       룸 2×2        4스테이지   압력 비용 첫 체험      어려움 시작
챕터 3 — 해안 용암길     룸 2×2~3×2   4스테이지   영속성과 되돌림 불가   어려움
챕터 4 — 화산 중심부     룸 3×3        4스테이지   자원 교차 / 복합 잠금  고난이도
최종 스테이지             룸 3×3 특수   1스테이지   전 어휘 통합           고난이도
```

총 17스테이지. 각 스테이지는 하나의 핵심 질문을 가진다.

## 챕터 1 — 감귤 과수원

**테마**: 각 어휘가 무엇인지 배운다. 실수해도 소프트락보다는 체력 낭비로 처벌.

룸 그리드: 1×3~1×5 (선형). 방 배치는 직선이라 길 잃을 일이 없다.

---

### 1-1 — 처음 만남 (5룸 튜토리얼, 구현 완료)

**핵심 질문**: 이 캐릭터는 무엇을 할 수 있는가?

```
[구덩이] → [압력 버튼+돌문] → [폭탄+나무문] → [물+열쇠] → [땅굴+상자]
```

각 방이 하나의 어휘를 독립적으로 가르친다. 특히 방 [1,0]의 압력 버튼은
"문을 통과한 뒤 돌을 치우면 문이 닫힌다"를 몸으로 경험하게 한다.

---

### 1-2 — 첫 선택 (돌의 두 용도)

**핵심 질문**: 이 돌을 구덩이에 쓸 것인가, 버튼에 쓸 것인가?

룸 구성: 1×2

**방 [0,0]**:
- 돌 3개. 버튼 2개(모두 눌러야 돌문 개방). 구덩이 1개.
- 돌문은 방 [1,0]으로 가는 유일한 길.
- 구덩이 위치: 키 회수 후 돌아오는 동선 단축 경로에 있음.

**방 [1,0]**:
- 물 호수, 열쇠(물 속), 상자.
- 입구에서 상자까지 수영 없이는 우회 경로가 길다.

**퍼즐 구조**:
- 돌 2개를 버튼에 → 돌문 열림 → 방 [1,0]에서 열쇠+상자 클리어.
- 남은 돌 1개로 구덩이를 메우면 귀환 경로가 6칸 단축됨.
- 구덩이를 안 메우면 거북이로 수영 후 긴 우회 → 체력 압박.

**배울 것**: 돌은 동시에 두 곳에 쓸 수 없다. 어디에 투자할지 결정해야 한다.

**소프트락 없음**: 버튼에서 돌을 떼면 문이 닫히지만 돌문 앞까지 갈 수 있으므로
실수해도 돌을 다시 올려 재시도 가능.

---

### 1-3 — 물길의 순서

**핵심 질문**: 거북이가 어느 방향으로 나와야 하는가?

룸 구성: 2×1

**방 [0,0]**:
- 물 호수가 방을 대각으로 가로지름. 열쇠는 물 한가운데.
- 물 밖에서 물 진입 지점이 2개(좌, 우).
- 우측으로 수영해 나오면 방 [1,0] 방향으로 이미 5칸 전진.
- 좌측으로 나오면 우회 경로, 체력 10 낭비.

**방 [1,0]**:
- 나무벽 1칸(토끼 전용 통과). 상자는 나무벽 너머.

**퍼즐 구조**:
- 거북이로 물을 건너되 **우측 출구**로 나와야 효율적.
- 토끼로 나무벽 통과.
- 순서: 거북이 먼저 열쇠 획득 → 캐릭터 전환 → 토끼로 나무벽 통과 → 상자.

**배울 것**: 물 진입/출구 방향이 이후 캐릭터 전환 위치를 결정한다.
위치가 곧 자원이다.

---

### 1-4 — 순서가 열쇠다

**핵심 질문**: 세 캐릭터를 어떤 순서로 써야 하는가?

룸 구성: 2×2

**방 [0,0]**:
- 돌 1개. 버튼 1개. 돌문(방 [1,0] 입구).
- 나무벽 1칸이 버튼 주변 경로에 놓여 있음.
  고양이가 돌을 버튼 위로 밀면 나무벽과의 위치 관계로 인해
  토끼의 땅굴 출구 지점이 막힘.

**방 [1,0]**:
- 돌문으로 진입. 귤 3개.
- 남쪽 출구 → 방 [1,1].

**방 [0,1]**:
- 물 호수. 열쇠(물 속).
- 동쪽 출구 → 방 [1,1].

**방 [1,1]**:
- 상자.

**퍼즐 구조 (함정)**:
- 고양이로 먼저 돌 → 버튼 → 돌문 열림 → 방 [1,0]→[1,1] 도달.
- 열쇠 없음. 방 [0,1]에 접근하려면 나무벽 통과(토끼 땅굴)가 필요.
- 하지만 이미 돌이 버튼 위에 있어 토끼의 땅굴 출구를 막고 있음 → 소프트락.
  (땅굴은 출구 지점이 비어 있어야 성립한다.)

**정답 순서**:
1. 토끼 먼저 나무벽 통과 → 방 [0,1] 접근 경로 확보 (출구 지점이 아직 비어 있음).
2. 고양이로 돌 → 버튼 → 돌문 개방.
3. 방 [1,0]→[1,1] 진입.
4. 방 [0,1] 진입 → 거북이 수영 → 열쇠.
5. 방 [1,1] → 상자 클리어.

**배울 것**: 캐릭터 순서가 공간 경로를 결정한다.
어떤 캐릭터를 먼저 쓰면 다른 캐릭터의 이동 경로가 봉쇄된다.

## 챕터 2 — 귤밭 창고

**테마**: 압력 버튼이 비용이 된다. 자원을 어디에 묶어둘지 고민해야 한다.

룸 그리드: 2×2 고정.

---

### 2-1 — 귤밭 창고 (구현 완료)

**핵심 질문**: 두 버튼에 두 돌을 묶을 수 있는가?

구현 상태 참조. 돌 2개, 버튼 2개, 물+열쇠, 토끼 통로.

---

### 2-2 — 하나씩 열어야 한다 (구현 완료)

**핵심 질문**: 돌이 부족하다면 두 문을 어떤 순서로 열어야 하는가?

룸 구성: 2×2

**방 [0,0]**:
- 돌 3개 (R1, R2, R3).
- 버튼 그룹 A (A1 + A2 둘 다 눌러야 돌문 a 개방 → [1,0]).
- 버튼 그룹 B (B1 + B2 둘 다 눌러야 돌문 b 개방 → [0,1]).
- 배치 제약: R2는 A2에만, R3는 B2에만 도달 가능.
  R1은 A1 또는 B1 중 하나에만 배치 가능 (두 버튼 간 거리가 멀어 동시 불가).

**방 [1,0]**:
- 귤 3개. 하단 출구 → [1,1].

**방 [0,1]**:
- 물 호수. 열쇠(물 속). 우측 출구 → [1,1].

**방 [1,1]**:
- 상자.

**퍼즐 구조 (함정)**:
- R1→A1, R2→A2, R3→B2 배치 시도: 그룹 A는 활성(문 a 열림), 그룹 B는 B1 비어 있어 비활성.
- R1을 A1에서 B1으로 옮기면 그룹 A 비활성 → 문 a 닫힘.
  R1+R3로 그룹 B 활성 → 문 b 열림.
- 두 문을 동시에 열 수 없다. R1이 A와 B 사이에서 공유되어야 한다.

**정답**:
1. R1→A1, R2→A2 → 문 a 열림 → [1,0] 진입, 귤 수집.
2. [1,0] → [0,0] 귀환 (문 a 여전히 열림 — 돌이 그대로).
3. R1을 A1 → B1으로 이동 → 문 a 닫힘.
4. R3→B2 → 문 b 열림 → [0,1] 진입.
5. 거북이 수영 → 열쇠 → [1,1] → 상자.

**핵심**: R1이 두 그룹의 '공유 돌'이다. 빌려 쓰고 돌려줘야 하는데,
돌리는 순간 앞 문이 닫힌다. 귀환 타이밍이 정답의 일부다.

---

### 2-3 — 폭탄 순서 (구현 완료)

**핵심 질문**: 폭탄이 길을 열어준 직후, 어떤 순서로 행동해야 하는가?

룸 구성: 1×3 (선형)

**방 [0,0]**:
- 폭탄 A (range 2). 우측 range 2 내에 나무문 X (방 [1,0] 진입구).
- A 기폭 → X 파괴 → 방 [1,0] 진입 가능.

**방 [1,0]**:
- 폭탄 B (range 3). 우측 blast 내에 나무벽 T + 그 너머에 구덩이 P와 나무문 Y.
  - B 기폭 → T 파괴(바닥이 됨, 버튼 구역 접근 개방) + Y 파괴(방 [2,0] 진입구 개방).
  - T가 사라지면 P(구덩이)가 이동 경로에 노출됨 — 건너려면 돌로 메워야 함.
- 돌 R1: T 서쪽에 위치. T 파괴 전까지 T 너머로 밀 수 없음.
- 버튼 B1: T 동쪽(T 파괴 후 접근 가능). 돌문 D1 → 귤 클러스터 구역.
- 구덩이 P: B1 동쪽, 방 [2,0] 입구 직전.

**방 [2,0]**:
- 물 + 열쇠 + 상자. 거북이 수영 필수.

**올바른 순서**:
1. A 기폭 → X 파괴 → 방 [1,0] 진입.
2. B 기폭 → T + Y 파괴.
3. R1을 동쪽으로 밀어 B1 위에 올림 → D1 개방 → 귤 구역 진입, 귤 수집.
4. R1을 B1에서 한 칸 더 동쪽(P)으로 밀기 → P 소멸. D1 닫힘.
5. 방 [2,0] 진입 → 거북이 수영 → 열쇠 → 상자.

**함정**: B 기폭 후 R1을 B1을 그냥 지나쳐 바로 P로 밀면 → R1이 구덩이에 소멸 →
B1 활성 불가 → 귤 미수집 → [2,0] 수영에 체력 부족 → 소프트락.

**배울 것**: 폭탄이 열어준 구역에 돌의 두 용도(버튼 vs 구덩이)가 동시에 등장한다.
순서를 틀리면 돌이 영구 소멸하여 되돌릴 수 없다.

---

### 2-4 — 다른 방의 버튼 (구현 완료)

**핵심 질문**: 저 방의 돌이 이 방의 문을 열 수 있는가?

룸 구성: 2×2

**크로스룸 버튼 그룹**: 방 [0,1]의 버튼 B1 → 방 [0,0]의 돌문 D (동쪽 출구 → [1,0]).

**방 [0,0]**:
- 돌문 D (닫힘). D는 방 [0,1]의 버튼 그룹이 제어. D 너머 → 방 [1,0].

**방 [0,1]**:
- 돌 R1. 버튼 B1. 물 호수. 열쇠(물 속).

**방 [1,0]**:
- 귤. 하단 출구 → [1,1].

**방 [1,1]**:
- 상자.

**퍼즐 구조**:
- 방 [0,0] 시작. D 닫혀 있어 [1,0] 진입 불가.
- 남쪽으로 방 [0,1] 진입 → R1을 B1에 올림 → 방 [0,0]의 D 개방.
  돌은 B1에 남아 있으므로 D는 플레이어가 다른 방에 있어도 계속 열려 있음.
- 방 [0,0]으로 복귀 → D 통과 → [1,0] 귤 수집 → [1,1].
- [1,1] 서쪽 → [0,1] 재진입 → 거북이 수영 → 열쇠 → [1,1] → 상자.

**배울 것**: 압력 버튼은 돌이 올라가 있으면 플레이어가 어디 있든 문을 유지한다.
버튼이 "다른 방"에 있어도 원리는 같다. 크로스룸 버튼 그룹이 이 원리의 첫 응용이다.

---

## 챕터 3 — 해안 용암길

**테마**: 되돌릴 수 없는 선택. 폭탄 소모, 구덩이 소멸, 땅굴 위치가 이후를 결정.

룸 그리드: 2×2~3×2. 룸 수 증가로 미답지 탐색 부담 시작.

---

### 3-1 — 화산 귤길 (09 설계 문서 참조)

**핵심 질문**: 어떤 문을 영구 파괴하고 어떤 문을 압력으로 열어야 하는가?

```
[A 폭탄 전실] --나무문--> [B 압력실] --돌문--> [C 물+열쇠+상자]
```

- 나무문: 폭탄으로 영구 제거.
- 돌문: 압력 버튼 유지 중에만 통과 가능.
- C의 물속 열쇠와 같은 방 상자 → 수영 후 이동.
- 돌이 버튼에서 벗어나는 순간 C로 가는 문이 닫힘.

---

### 3-2 — 땅굴이 동선을 바꾼다 (구현 완료)

**핵심 질문**: 토끼가 어디에 구멍을 내느냐가 나머지 퍼즐을 결정한다.

룸 구성: 2×2

**핵심 아이디어**:
- 토끼 땅굴 구멍 = 이후 토끼 전용 통로. **다른 캐릭터는 진입 불가**.
- 토끼가 방 [0,0]의 나무벽 A를 뚫어 구멍 냄 → 방 [1,0] 입구가 토끼 전용이 됨.
- 이 상태에서 거북이는 방 [1,0]에 진입 불가.
- 방 [1,0]에는 거북이가 필요한 물+열쇠가 있음.
- **정답**: 토끼는 나무벽 B(다른 위치)를 뚫어야 한다. A를 뚫으면 소프트락.

**배울 것**: 땅굴의 위치는 "나만 지나갈 수 있는 구멍"을 만든다.
다른 캐릭터의 경로를 봉쇄하지 않는 위치를 선택해야 한다.

---

### 3-3 — 귤이 전략이다 (구현 완료)

**핵심 질문**: 귤을 건너뛰면 정말 막히는가?

룸 구성: 3×1

**방 [0,0]**:
- 돌 1개, 버튼 1개. 돌문 → [1,0].
- 귤 없음.

**방 [1,0]**:
- 분기: 상단 경로(짧음, 나무벽 1칸 — 토끼 필요), 하단 경로(우회, 귤 클러스터).
- 귤은 하단 경로에만 존재. 획득 시 +30 HP. 상단으로 가면 귤 없음.

**방 [2,0]**:
- 물 호수. 열쇠(물 속). 상자.
- 열쇠까지 수영 8칸 필수.

**수치 설계 (이진 판정)**:
- 시작 HP: 50. 귤 없이 [2,0] 수영 직전까지의 최단 경로: 약 44 스텝 소모.
  남은 HP ≈ 6. 수영 8칸 불가 — 체력 부족으로 게임오버.
- 귤 클러스터 경유 시: 추가 이동 약 10스텝 (−10) + 귤 +30 = 실질 +20 HP.
  남은 HP ≈ 26. 수영 8칸 가능.

**함정**: 상단(빠른) 경로를 택하면 방 [2,0] 물 앞에서 체력이 바닥난다.
귤은 선택지가 아니라 **수영을 가능하게 하는 유일한 자원**이다.
"빠른 길이 옳다"는 직관이 틀린 것을 직접 확인해야 한다.

**배울 것**: 귤은 장식이 아니라 체력 예산이다. 경로를 선택하기 전에 "이 길을 가면 나중에 뭘 할 수 있는가"를 계산해야 한다.

---

### 3-4 — 세 관문 (구현 완료)

**핵심 질문**: 세 개의 잠금이 순서를 강제한다.

룸 구성: 3×2 (6룸)

세 개의 연속 퍼즐이 서로 의존한다:
1. **방 [0,0]→[1,0]**: 폭탄으로 나무문 제거 → 내부 구조 노출.
2. **방 [1,0]→[2,0]**: 폭탄이 열어준 영역에 버튼 있음 → 돌문 개방.
3. **방 [0,1]→[1,1]→[2,1]**: 물 + 열쇠 + 상자.

폭탄 1은 나무문을 열고, 이 과정에서 나무 장애물도 제거된다. 이 나무 장애물이
제거되어야만 돌을 버튼까지 밀 수 있다.
즉: 폭탄 순서 → 동선 개방 → 버튼 활성 → 열쇠 방 입구 개방 → 클리어.

---

## 챕터 4 — 화산 중심부

**테마**: 자원이 교차한다. 돌 하나가 두 역할을 해야 한다.

룸 그리드: 3×3 (9룸). 맵이 커서 미니맵 활용 필수.

---

### 4-1 — 두 개의 저울 (구현 완료)

**핵심 질문**: 왼쪽과 오른쪽을 동시에 열 수 없다면 어떤 순서로?

버튼 그룹 left_gate, right_gate. 돌이 부족해 동시 활성 불가.
토끼 땅굴 숏컷으로 재배치 효율화.

룸 구성: 3×3 그리드의 십자형 5룸.

구현 구조:
- 아래 방 [1,2]에서 시작해 중앙 방 [1,1]로 진입.
- 중앙 방에는 돌 1개, 버튼 2개, 좌우 게이트 2개가 있다.
- 돌을 왼쪽 버튼에 올려 [0,1] 열쇠 방을 먼저 열고, 열쇠를 얻은 뒤 중앙으로 복귀한다.
- 같은 돌을 오른쪽 버튼까지 다시 밀어 [2,1] 상자 방을 연다.
- 중앙의 짧은 나무벽은 금향이 땅굴로 넘을 수 있어 재배치 동선을 줄인다.

---

### 4-2 — 돌 3개, 버튼 4개 (구현 완료)

**핵심 질문**: 어떤 버튼은 돌 1개를 공유할 수 있는가?

룸 구성: 3×3 (일부만 활성)

**버튼 그룹**:
- 그룹 A: 버튼 2개 → 돌문 a.
- 그룹 B: 버튼 2개 → 돌문 b.
- 사용 가능한 돌: 3개.

**퍼즐 구조**:
- 그룹 A의 버튼 하나와 그룹 B의 버튼 하나가 같은 경로에 있음.
- 돌 1개를 공유할 수 있는 배치가 존재.
- 즉 그룹 A에 돌 2개(A1, A2), 그룹 B에 돌 2개가 필요하지만,
  실제로 필요한 돌은 A1을 나중에 B1 위치로 재배치하면 3개로 충분.
- 다만 A1을 빼는 순간 그룹 A 비활성 → 돌문 a 닫힘.
- 돌문 a를 통과한 뒤 a가 닫혀도 다른 경로로 귀환 가능한지가 핵심.

구현 구조:
- 중앙 방 [1,1]에 돌 3개와 버튼 4개를 배치했다.
- shared_a_gate는 아래쪽 버튼 2개를 눌러 위쪽 열쇠 방 [1,0]을 연다.
- shared_b_gate는 위쪽 버튼 2개를 눌러 오른쪽 상자 방 [2,1]을 연다.
- 왼쪽 아래 버튼에 올린 돌을 위쪽 버튼까지 밀어 올려 공유 돌로 재사용한다.
- 첫 게이트는 공유 돌을 빼는 순간 닫히지만, 열쇠를 얻고 중앙으로 복귀한 뒤 진행하므로 다음 목표가 명확하게 바뀐다.

---

### 4-3 — 연쇄 잠금 (구현 완료)

**핵심 질문**: 열쇠를 얻은 뒤 그 돌을 상자 문 버튼에도 써야 한다.

룸 구성: 3×3

**구조**:
- 방 A: 돌 2개. 버튼 그룹 → 방 B 돌문.
- 방 B: 물. 열쇠(물 속). 버튼 1개 → 방 C 나무문.
- 방 C: 상자.
- 방 B의 버튼은 방 A의 돌 1개가 도달해야 누를 수 있음.
  (방 A에서 돌을 방 B까지 경로 이동 가능한 구조)

**정답**:
1. 방 A 돌 2개로 방 B 돌문 개방 → 방 B 진입.
2. 방 B 내부로 방 A의 돌 1개를 끌어와 방 B 버튼에 올림 → 방 C 나무문 개방.
3. 거북이로 수영 → 열쇠 획득.
4. 방 C로 이동 → 상자 클리어.

구현 구조:
- 현재 규칙에서는 돌이 방 경계를 넘지 않으므로, 중앙 방 [1,1] 안에서 공유 돌을 재사용하는 방식으로 구현했다.
- chain_key_gate는 중앙 방의 버튼 2개로 위쪽 물/열쇠 방 [1,0]을 연다.
- 청귤이로 물길 중앙의 열쇠를 얻은 뒤 중앙 방으로 복귀한다.
- 열쇠 방을 열던 돌 하나를 빼서 chain_chest_gate 버튼으로 옮기면 오른쪽 상자 방 [2,1]이 열린다.
- 첫 게이트가 다시 닫혀도 목표가 상자 방으로 전환되므로, “잠금 해제 후 자원 회수” 흐름이 명확하다.

---

### 4-4 — 화산 방어 (구현 완료)

**핵심 질문**: 모든 것이 한 번에 맞아야 한다.

룸 구성: 3×3 (9룸 전부 활성)

전 어휘 통합:
- 폭탄 순서 (2개) — 잘못 기폭하면 버튼 경로 차단.
- 압력 버튼 그룹 2개 — 돌이 양쪽에 부족.
- 물 + 거북이 전환 잠금 — 물에서 나오는 위치 선택.
- 토끼 땅굴 — 재배치 숏컷.
- 구덩이 1개 — 돌을 여기 쓰면 버튼 1개 영구 포기.

**난이도 기준**: α ≈ 1.2 (여유율 최소). 최적 동선이 아니면 체력 부족.

구현 구조:
- 3×3 룸 전체를 데이터에 두고, 실제 해법은 중앙 [1,1] → 상단 [1,0] → 오른쪽 [2,1] 축으로 진행한다.
- 중앙 방에서 돌 하나를 구덩이에 밀어 넣어 이동 공간을 확보한다.
- 돌 2개로 volcano_key_gate를 열고, 상단 방으로 진입한다.
- 상단 방에서는 폭탄으로 나무문을 파괴한 뒤 청귤이로 물길의 열쇠를 회수한다.
- 중앙으로 돌아온 뒤 금향이 땅굴로 짧은 나무벽을 넘어 돌 재배치 위치로 복귀한다.
- key_gate에 쓰던 돌 하나를 빼서 volcano_chest_gate 버튼으로 옮기면 오른쪽 상자 방이 열린다.
- 시작 HP 80, 귤 0개, 해법 잔여 HP 4로 챕터 보스 난이도를 만든다.

---

## 최종 스테이지 — 화산 정상 (구현 완료)

**핵심 질문**: 이 섬을 구할 수 있는가?

룸 구성: 3×3 특수 (중앙 룸이 용암 타일로 특수 처리)

**서사 장치**: 이 스테이지만 BGM이 다름. 클리어 시 저주 말뚝(`o_stake`) 제거 연출.

**퍼즐 구조**:
- 중앙 룸: 물 대신 용암(규칙은 물과 동일, 시각만 다름). 거북이만 통과.
- 열쇠 방: 거북이 전환 잠금이 걸리는 위치에 열쇠 → 잠금 해제 전까지 캐릭터 전환 불가.
- 압력 버튼 그룹 3개 → 상자 방 삼중 잠금.
- 폭탄 1개 — 핵심 경로에 배치. 남겨두거나 써야 할지 선택.
- 해법: 토끼 땅굴로 버튼 그룹 중 하나 도달 → 고양이로 돌 2개 재배치 →
  거북이 전환 잠금 해제 후 열쇠 획득 → 상자.

구현 구조:
- 스테이지 ID는 `5-1`, 화면 표시는 `STAGE 5-1 / 화산 정상`.
- 3×3 룸 전체를 데이터에 두고, 중앙 [1,1]·상단 [1,0]·오른쪽 [2,1]을 핵심 동선으로 사용한다.
- 중앙 방에는 구덩이 1개, 돌 4개, 버튼 4개, 상자 방을 막는 삼중 돌문을 배치했다.
- 돌 하나는 구덩이를 메우는 데 쓰고, 두 돌은 summit_key_gate를 열어 상단 정상 방으로 진입한다.
- 상단 방에는 폭탄, 나무문, 물길, 열쇠, 서사 장식 `stake`가 있다. 현재 렌더러에서는 `stake`가 규칙에 간섭하지 않는 데이터 장식이다.
- 정상의 물길은 용암 역할을 하지만, 현재 엔진에서는 `~` 물 타일을 사용해 청귤이 전용 통과 규칙을 그대로 따른다.
- 열쇠를 얻고 돌아온 뒤 금향이 땅굴로 중앙 나무벽을 넘어 공유 돌을 마지막 버튼으로 재배치한다.
- summit_chest_gate는 버튼 3개가 모두 눌려야 오른쪽 삼중 돌문을 열며, 오른쪽 방의 상자를 열면 게임 클리어.
- 시작 HP 92, 귤 0개, 해법 잔여 HP 8로 최종 스테이지 압박감을 유지한다.

## 원작 반영 재작업 지시 (2026-07-16)

구현 완료 표시된 스테이지들은 규칙은 유지하되 아래 재작업 대상이다. 각 항목은 솔버(validate-all.js) 전수 재검증을 통과해야 반영된다.

### 대상 1 — 전 스테이지 레이아웃 패스

- 모든 방을 레이아웃 원칙 5개(위 절)로 재검사한다. 통행 영역이 직사각형인 방은 담장 클러스터와 꺾인 담장으로 재타일링한다.
- 재타일링은 해법 스텝 수를 바꾸므로 `solutionPath`와 체력 수치를 함께 갱신한다.
- 1-1의 첫 방만 예외 (튜토리얼 가독성).

### 대상 2 — 챕터 2~4 난이도 상향 패스

- 난이도 곡선 표의 α 목표에 맞춰 시작 HP와 귤 배치를 조인다.
- 여유 돌을 제거한다 (돌 수 = 해법 필요 수).
- 챕터 2 이후 모든 힌트 오브젝트를 제거한다.
- 각 스테이지에 함정(눈에 보이는 소프트락 또는 체력 파산 경로)이 최소 1개 있는지 확인하고, 없으면 추가한다.
- 재배치 회귀(돌-버튼 재사용) 없이 풀리는 스테이지는 버튼 그룹을 재구성한다.

### 대상 3 — 신규 스테이지 삽입 (11 문서 §4)

- 1-5 징검다리 (수영 거리 제한 소개 — 챕터 1이므로 쉬움 유지)
- 3-5 타버린 과수원 (폭발 부수 피해)
- 4-5 저수지 밸브 (밸브 + 수영 제한 + 숏컷 상실)

### 검증 기준

- 솔버 출력의 잔여 HP가 챕터별 α 목표 범위에 들어가는가.
- "우회 최단해"가 재배치 회귀를 생략하지 못하는가 (09 §6).
- 레이아웃 직사각형 검사를 통과하는가.

## 스테이지별 핵심 질문 요약

| 스테이지 | 핵심 질문 | 신규 어휘 조합 |
|---|---|---|
| 1-1 | 이 캐릭터는 무엇을 할 수 있는가? | 각 어휘 독립 소개 |
| 1-2 | 이 돌을 구덩이에 쓸 것인가, 버튼에 쓸 것인가? | 돌 × (구덩이 vs 버튼) |
| 1-3 | 거북이가 어느 방향으로 나와야 하는가? | 물 전환 잠금 × 위치 |
| 1-4 | 캐릭터를 어떤 순서로 써야 하는가? | 캐릭터 순서 × 공간 봉쇄 |
| 2-1 | 두 버튼에 두 돌을 묶을 수 있는가? | 압력 × 돌 2개 |
| 2-2 | 돌이 부족하면 두 문을 어떤 순서로? | 그룹 버튼 × 돌 공유 |
| 2-3 | 폭탄이 열어준 구역에서 돌을 어떻게 쓰나? | 폭탄(영구) × 돌(버튼 vs 구덩이) |
| 2-4 | 저 방의 돌이 이 방의 문을 열 수 있는가? | 크로스룸 압력 버튼 |
| 3-1 | 영구 파괴 vs 압력 개방 — 어느 것을? | 폭탄(영구) × 압력(임시) |
| 3-2 | 땅굴 위치가 다른 캐릭터를 막는가? | 땅굴 × 캐릭터 진입 제한 |
| 3-3 | 귤을 건너뛰면 정말 막히는가? | 귤(체력) × 이진 경로 |
| 3-4 | 세 잠금이 순서를 강제한다 | 폭탄 → 압력 → 수영 연쇄 |
| 4-1 | 왼쪽/오른쪽을 동시에 열 수 없다 | 그룹 2개 × 돌 부족 |
| 4-2 | 돌 1개를 두 그룹이 공유할 수 있는가? | 공유 돌 × 귀환 경로 |
| 4-3 | 열쇠 방 버튼도 돌로 눌러야 한다 | 돌 이동 × 연쇄 잠금 |
| 4-4 | 전 어휘를 동시에 최적화할 수 있는가? | 통합 |
| 최종 | 이 섬을 구할 수 있는가? | 삼중 잠금 + 전 어휘 |
