# Source Materials Upload Folder

Put original or reference source material here.

Use this folder for files that should be preserved as references:

- original feature-phone source archives
- old build notes
- device specifications
- original rule documents
- screenshots of the old game
- reference code that is not yet part of the new implementation

When we decide what code should become part of the playable modern game, it can be moved or adapted into `../../src/`.
